Detailed Description

TTK morseSmaleComplex2D processing package.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
February 2017.

MorseSmaleComplex2D is a TTK processing package that takes a scalar field on the input and produces a scalar field on the output.
