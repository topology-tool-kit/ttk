Detailed Description

TTK VTK-filter which generates a texture map from one or two point data scalar fields.

Author
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
November 2014.
Parameters
InputInput data set (vtkDataSet)
OutputOutput data set with texture coordinates (vtkDataSet)

This filter is useful to convert scalar fields to texture coordinates or to generate texture-based level lines out of a single scalar fields.

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.
